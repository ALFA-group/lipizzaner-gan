Quickstart

Use this web application to review the logs Lipizzaner writes into it's log server.

Usage:

  • Select an experiment on the left (by default, the latest is already selected).
  • Change the iterations displayed in the grid with the slider (default: latest iteration).
  • Click a cell to display the whole experiment progress for the respective node.

Changing the log server is currently not supported/will be added in future releases.
